# Directorio de Emprendimientos Plant-Based

## Descripción
Landing page "Próximamente" para futuro directorio de negocios veganos y plant-based en CDMX, con sistema de registro temprano vía mailto.

## Funcionalidades actuales
- Hero section con llamada a la acción
- Grid de categorías placeholder
- Tarjetas de negocio skeleton para mostrar la visión
- Sistema de contacto pre-rellenado vía mailto
- Sección de beneficios para early adopters

## Componentes
- `BusinessCard`: Componente reutilizable para futura implementación
- Grid responsive de categorías y placeholders
- Sistema de contacto integrado

## Extensibilidad futura
- Base de datos de negocios
- Sistema de filtros y búsqueda  
- Perfiles detallados de emprendimientos
- Sistema de valoraciones y reseñas
